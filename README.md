# SAP BASIS Linux Practice Lab

This repository demonstrates hands-on Linux system administration skills
aligned with real-world SAP BASIS operations used in enterprise environments.

## Objective
To simulate SAP BASIS administration tasks such as:
- Linux user & group management
- SAP directory structure setup
- Permission and ownership handling
- System monitoring and log analysis
- Service management

## Environment
- OS: Ubuntu 22.04 LTS (SAP-like Linux simulation)
- Virtualization: Oracle VirtualBox
- Tools: Bash, systemctl, journalctl

## Skills Demonstrated
- Linux administration for SAP systems
- SAP server directory layout understanding
- User, group, and permission management
- Troubleshooting via logs
- Enterprise documentation practices

## Disclaimer
This project does not contain SAP software or proprietary code.
It simulates SAP BASIS administrative tasks for learning purposes only.
